The Warm
Navigator.

z for Finder is rupa's terminal directory-jumper, reborn as a macOS app. One hotkey, a few letters, and you're there — ranked by where you actually work, not by what a bloated index guesses. The promise isn't features. It's it works every single time, in 50ms.

01

What the spikes told us

Three throwaway probes, run against a real Mac before a line of UI was written. The point was to kill the idea cheaply if the core signal wasn't real. It survived — and got sharper.

Spike 0 · falsified → rescued

Use-count was a trap. Recency wasn't.

The naïve kMDItemUseCount rollup failed twice: the mdls CLI couldn't read metadata at all (a broken Spotlight index, since rebuilt), and counting files surfaced photo archives, not workplaces. Recency via the framework was the real signal.

Spike 1 · pass

The framework path is solid & fast.

NSMetadataQuery returned 332/332 files with a readable kMDItemLastUsedDate in ~1.3s incl. compile. Where the flaky CLI failed, the API a real app would use works perfectly. Sub-50ms in-app is well within reach.

Spike 2 · pass

Tuned ranking reads like your life.

Blend recency × √(useCount), roll up to the nearest .git root, down-weight transient dirs. The top of the list became real work — client decks, music projects, active repos, finances — not Downloads.

Land

Finder, default app, or — one keystroke away — editor & terminal at that path.

doesLearns from you, silently

Frecency from native signals now; live Finder/editor observation later. It gets more yours over time.

doesProject-aware

Rolls deep paths up to the git repo you think in, not the leaf folder a file happens to live in.

doesReliable by design

Small, hand-tuned, place-only index. The whole product is the absence of "it didn't find it."

won'tLaunch apps

Spotlight & Raycast own that. Adding it dilutes the focus and the reliability promise.

won'tOrchestrate contexts

Opening three apps at once is Bunch's job — and where "every time, 50ms" goes to die.

won'tBe everything

No web, no math, no AI chat. A navigator. Saying no is the entire strategy.
